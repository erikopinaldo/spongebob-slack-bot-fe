Spongebob Meme Text Generator

A Slack app that converts text like "This is some text" to "tHiS iS SOMe TExT".

How to use

This app is activated with a slash command. You can type the following into the message input field and send it as a message:

/spongebobtext This is some text

Output:

tHiS iS SOMe TExT
